Gerador de QR Code Simples
Este é um projeto simples de um site que gera um QR Code a partir de um link fornecido pelo usuário. O projeto foi desenvolvido usando HTML, CSS e JavaScript, com a biblioteca externa qrcode.js para gerar os QR Codes.

Funcionalidades
O usuário insere um link no campo de texto.

Ao clicar no botão "Gerar QR Code", um QR Code é gerado e exibido na tela.

O QR Code pode ser escaneado por qualquer leitor de QR Code para redirecionar para o link inserido.

Como Usar
Clone este repositório ou faça o download dos arquivos.

Abra o arquivo index.html no seu navegador.

Insira um link no campo de texto e clique em "Gerar QR Code".

O QR Code será exibido na tela e poderá ser escaneado.
